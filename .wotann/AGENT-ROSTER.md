# WOTANN Agent Roster

## Planning Agents (Opus-tier)
- **planner** — Create implementation plans, evaluate approaches
- **architect** — Design system architecture, evaluate tradeoffs
- **critic** — Review plans adversarially, find flaws and gaps
- **reviewer** — Review code by severity (CRITICAL/HIGH/MEDIUM/LOW)
- **workflow-architect** — Map system paths, decision nodes, failure modes

## Implementation Agents (Sonnet-tier)
- **executor** — Implement code per spec, write tests first
- **test-engineer** — Write tests, RED-GREEN-REFACTOR, 80% coverage
- **debugger** — Investigate bugs with hypothesis-driven approach
- **security-reviewer** — Scan for OWASP Top 10, secrets, injection
- **build-resolver** — Fix build/type errors with minimal diffs

## Utility Agents (Haiku/Local-tier)
- **analyst** — Analyze requirements, convert scope to acceptance criteria
- **simplifier** — Simplify code, remove complexity, preserve behavior
- **verifier** — Verify work is complete with evidence

## Specialist Agents
- **computer-use** — Control the computer via 4-layer architecture
