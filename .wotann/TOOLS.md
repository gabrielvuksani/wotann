# WOTANN Tools

## Core (always loaded)
- **Read** — Read files with line numbers
- **Write** — Create or overwrite files
- **Edit** — String replacement in files
- **Glob** — File pattern matching
- **Grep** — Content search (ripgrep)
- **Bash** — Shell command execution
- **LSP** — Language server operations

## Standard (loaded by default)
- **WebSearch** — Web search
- **WebFetch** — Fetch URL content
- **Agent** — Spawn subagent
- **Skill** — Invoke a skill
- **ComputerUse** — Screen control (4-layer)

## Enhanced
- **HashlineEdit** — Edit by content hash (10x more reliable)
- **SymbolEdit** — Edit by symbol name via LSP
- **SmartRename** — LSP rename + grep search
- **SmartCommit** — Auto-split into conventional commits

## MCP Servers
- context7 — Library documentation
- lightpanda — Fast headless browser
- playwright — Browser automation
- qmd — Precision retrieval (90% token savings)
