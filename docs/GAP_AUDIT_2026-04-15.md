### Hook system — semantic gaps

| Hook | Issue | Impact |
|---|---|---|
| **MemoryRecovery** | Loads WAL marker on SessionStart but only surfaces the tailContent LENGTH in the result message — the actual content is discarded. | Context recovery is cosmetic only |
| **ReadBeforeEdit** | `recentlyReadFiles` is a module-global `Set<string>`, not per-session; never evicted. | Cross-session leakage + unbounded growth |
| **CompletionVerifier** | Evidence regex trivially gameable: the literal word "pass" or "✓" anywhere in the final message bypasses the check. | Strict-profile verification is a paper tiger |
| **PostToolUse hooks** | Fire with `content: fullContent` — the AGENT'S response text, not the tool's actual output. | ResultInjectionScanner scans the wrong layer |
| **HookEngine.fireSync** | `catch {}` swallows all handler errors; returned HookResult's warnings aren't surfaced either. | Silent hook failures |
| **Stop event** | `runtime.close()` calls `fireSync` but discards the returned HookResult, so a blocking CompletionVerifier cannot actually prevent close. | Strict-profile Stop-blocking is a no-op |
| **PreCompactWALFlush** | `payload.content` for PreCompact is `${stage}:${removedMessages}` — a 20-30 char metadata label — not the actual compacted messages. | "Context survives compaction" claim is cosmetic |
| **DestructiveGuard patterns** | Missing `rm -r` (without -f), `git reset --mixed`, `truncate`, `shred`, `dd if=`, shell `>` redirects overwriting critical files | Pattern set too narrow |

`desktop-app/src/components/palette/CommandPalette.tsx` call
`tauriCommands.sendMessage("/session create")` etc. — raw strings, not

helper that generates `{method, params}` JSON before calling
`sendMessage`.
